# zyt-signal

A deliberately minimal public signal page for `LSB / ZYT`.

Visual identity: a cyan-violet cyber butterfly that fully illuminates when the two signals merge.
Interaction: the first click releases memory node 01; later clicks replay its signal effect.
Node art stays anonymous: abstract post comments flow into alternating two-sided message bubbles with no names or message text.

Public safety boundary: do not add personal names, photos, private messages,
precise dates, addresses, or credentials here.
